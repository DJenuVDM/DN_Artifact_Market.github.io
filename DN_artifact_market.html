What is DN Artifact Market?

DN Artifact Market is a Discord bot that connects Hades' Star players across every server into one shared artifact marketplace. Post a listing in your server and the entire network can see it. Search and you'll find listings from every other server too — not just your own.

No currency, no external websites. Pure artifact-for-artifact barter, all within Discord.

🌐 DN Artifact Market is hosted on a dedicated VPS — it runs 24/7, independent of any personal machine, and your data is not shared with any third party.

Trading Commands

/sell   Slash Command Prefix: !sell
Create a listing to sell artifacts. Specify type, level, quantity, boost %, and optional notes. Supports bundle mode (all-or-nothing) or separate mode (individual items). Listings can be server-only or global.
/buy   Slash Command Prefix: !buy
Create a "Wanted" listing specifying what you need AND what you'll pay. Other players see an "Accept & Trade" button to instantly start the swap.
/search   Slash Command
Browse the marketplace with powerful filters: artifact type, level range, boost %, listing type, bundle size, and multiple sort options. Results are paginated (10 per page).
/offer   Slash Command Button: 📨 Make Offer
Propose artifacts in exchange for a listing's items. For non-bundle listings, you can request specific quantities. The listing creator gets a DM notification with Accept/Counter/Decline buttons.
/counter   Slash Command Button
Counter an existing offer with different items. Unlimited counter-offer chains supported — negotiate until you agree.
/accept
Accept an offer or buy listing to start trading. Partial trades supported.
/decline
Decline an offer. The proposer is notified via DM.
/confirm
Confirm you received the items. Both must confirm to complete.
/cancel
Cancel a listing or an active trade. Partner is notified.
/rate
Rate your trade partner 1–10 after a completed trade.
/trade
View and switch between multiple active trades.
/dashboard
Personal profile: rating, rep, stats, active listings & offers.
/report
Report issues with multi-category flow. Chat logs preserved.

Cross-Server Groups

Groups let server admins create private networks of channels across different servers that share listings with each other exclusively. Instead of posting globally to every server, a group keeps trades within a trusted circle of communities.

🌐 Why Use a Group?

Perfect for alliances, friend groups, or partner communities who want a shared marketplace without opening up to the entire global network.

  • Create a group from any channel — it becomes the group's home channel automatically
  • Invite other servers by sharing your Group ID — they join with /group join
  • Listings in member channels are shared only within the group
  • Block servers you don't want joining — any existing channels from that server are removed instantly
/group create   Admin Only
Create a new group from the current channel. You become the group owner and get a shareable Group ID.
/group join
Add your current channel to an existing group using its Group ID. Requires group admin rights.
/group leave
Remove your current channel from a group. Requires group admin rights.
/group info
View all channels, admins, and blocked servers in a group.
/group mine
List all groups you have admin rights in, with your ownership status.
/group delete   Owner Only
Permanently delete a group and remove all its channels. Cannot be undone.
/group addadmin   Owner Only
Grant another user admin rights over the group so they can manage channels.
/group removeadmin   Owner Only
Revoke a user's admin rights from the group.
/group blockserver   Owner Only
Block a server from joining your group by its server ID. Any channels from that server are removed immediately.
/group unblockserver   Owner Only
Remove a server block, allowing that server to join the group again.

Reputation & Rating System

⭐ Ratings (1–10)
After every completed trade, both parties can rate each other on a 1–10 scale. Ratings are averaged and displayed on listings and profiles. One rating per trade — no spam.
🏆 Reputation Points
+1 reputation for every completed trade. Displayed on each user's dashboard alongside their items sold/bought counts.

Ratings are visible in search results, so buyers can see who they're trading with before making an offer.

Moderation

DN Artifact Market has a dedicated moderation team of trusted, appointed individuals. These are not your server's Discord administrators — they are specific people chosen to keep the marketplace fair and safe. Regular users do not have access to any moderation tools.

🚫 Bans
Moderators can ban users from the bot entirely. Banned users cannot create listings, make offers, or participate in trades.
📋 Reports
User-submitted reports are reviewed by the moderation team, who have access to full trade context including chat logs.
🏷️ Flags
Moderators can flag users with internal notes to track patterns of behaviour across multiple incidents.
🗑️ Data Removal
Moderators can delete trade records and process data deletion requests in line with our privacy policy.

Ban Appeal System

Banned users aren't left in the dark. When they try to use any trading command, they see an "Appeal Ban" button. Appeals are submitted via a short form, reviewed by the moderation team, and resolved with a reply sent directly to the user via DM.

  • Cooldown between appeals to prevent spam
  • Full back-and-forth conversation between the banned user and the moderation team
  • Outcome (kept or lifted) is communicated to the user via DM

Privacy & Data Protection

Built-in GDPR Compliance

  • Data deletion requests — users can request deletion of all their data right from /dashboard
  • Full deletion — admins or the bot can delete ALL user data in one atomic transaction: chat logs, ratings, reports, flags, appeals, offers, listings, transactions, and the user record
  • Auto-purge — chat logs and completed transactions are automatically deleted after 7 days
  • Monitoring notice — users are informed that trade messages may be recorded
  • Terms of Service & Privacy Policy — hosted at /terms and /privacy
